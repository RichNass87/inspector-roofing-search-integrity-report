# Roofing Search Integrity Demo

Public-safe Gradio app for classifying roofing search intent, homeowner questions, and page topics into practical trust, documentation, locality, repairability, and code-to-spec recommendations.

The demo is part of *The Roofing Search Integrity Report* source spine.

- DOI: https://doi.org/10.5281/zenodo.21040534
- GitHub: https://github.com/RichNass87/inspector-roofing-search-integrity-report
- Dataset: https://huggingface.co/datasets/InspectorRoofing/roofing-search-integrity-report
- Kaggle: https://www.kaggle.com/datasets/inspectorroofing/roofing-search-integrity-report
- Amazon paperback: https://www.amazon.com/dp/B0H6XXDL9X
- Paperback ISBN-13: 979-8184859057
- Inspector Roofing research page: https://inspector-roofing.com/roofing-search-integrity-report/

This public demo does not include private customer data, private claim files, direct Google result scraping, proprietary scoring rules, or ranking guarantees.
